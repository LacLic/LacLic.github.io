CH0103-0x00「基本算法」例题-最短Hamilton路径

Posted on 2021-03-09 Edited on 2021-04-01

题意

有 N 个地方，分别标记为 0 到 N-1，现在给定每两个地点之间的距离，要你求出最短的经过所有地方的路程。

其中，$N<=20$

思路

动态规划，记录两个状态

  1. 现在所在地
  2. 已经过的地方（使用二进制表示，经过的为 1，未经过的为 0）

当上述两个条件完全一致时，就可以直接调用之前 dfs 算出来的值
