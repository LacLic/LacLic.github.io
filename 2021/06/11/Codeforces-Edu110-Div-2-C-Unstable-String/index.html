Codeforces-Edu110(Div.2)C - Unstable String

Posted on 2021-06-11

题意

给定一个字符串，只包含 0 , 1 , ?

定义：0101010这种0或1交替出现的字符串为不稳定字符串

现在， ? 可以替换为 0 或 1

求：当前给定的字符串，能找出多少个子串为不稳定字符串

输入将给入t个case和t个字符串。

思路

dp题，从0开始，每向字符串添加一个字符，记录当前字符串子串能产生多少个不稳定字符串，并且当前产生的子串中，有多少个还能继续添加字符，并且其中结尾为 0 , 1 , ? 的各还有几个

状态转移方程：

first是当前能产生的不稳定子串个数，zero表示能继续添加字符且尾巴为0的子串个数，one和query同理

新添字符为 0

新添字符为 1

新添字符为 ?

然后更新一下尾巴字符的情况就好了
